# React application: Define Limit

## How to install?
#### 1. Clone the project
```git
git clone https://github.com/Wo0zZ1/limit-web.git
```
#### 2. Open the folder
``` bash
cd limit-web
```
#### 3. Install dependencies
```bash
npm install
```
#### 4. Start server
```bash
npm run dev
```
#### 5. [Open Browser](http://localhost:5173/)
